# 📱 Sistema de Consultas Mobile

Aplicação mobile desenvolvida com **React Native + Expo + TypeScript** para gerenciamento de consultas médicas.

---

## 🚀 Funcionalidades

* 📋 Cadastro de pacientes
* 👨‍⚕️ Cadastro de médicos
* 🏥 Cadastro de especialidades
* 📅 Gerenciamento de consultas
* 🔄 Controle de status das consultas

---

## 🛠️ Tecnologias utilizadas

* React Native
* Expo
* TypeScript
* Node.js

## ⚙️ Como rodar o projeto

### 1. Clone o repositório

```
git clone https://github.com/Murilo1704/sistemas-consultas-mobile.git
```

### 2. Acesse a pasta

```
cd sistemas-consultas-mobile
```

### 3. Instale as dependências

```
npm install
```

### 4. Execute o projeto

```
npx expo start
```
